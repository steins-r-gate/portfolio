User Experience Theory Applied

  • Visual Hierarchy - Clear importance levels through size, color, spacing
    Primary: Name & call-to-action buttons (large, high contrast)
    Secondary: Section headings (gradient accent)
    Tertiary: Body text (muted colors, readable size)
  • Progressive Disclosure - Information revealed as needed
    Contact modal shows detailed info only when requested
    Project descriptions concise with tech tags for details
    Scroll animations reveal content progressively
  • Consistency - Unified design language throughout
    Card design pattern repeated across sections
    Consistent spacing (1.5rem, 2rem, 3rem increments)
    Uniform hover effects (translateY, border-color changes)
  • Feedback & Affordance - Clear interactive indicators
    Cursor changes on hoverable elements
    Color changes indicate clickability
    Transform animations provide hover feedback
  • Accessibility (WCAG Guidelines)
    Color contrast: Text on backgrounds meets AA standards
    Keyboard navigation: Tab order logical, escape key works
    Semantic HTML: Proper heading hierarchy, landmark regions
